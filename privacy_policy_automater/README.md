# LinkedIn Auto Poster Privacy Policy

This folder is a standalone static privacy policy site for LinkedIn app setup.

## Deploy on Vercel

1. Push this folder to GitHub.
2. Create a new Vercel project.
3. Select this folder as the project root: `privacy_policy_automater`.
4. Use the default static site settings.
5. After deployment, use the Vercel URL as the Privacy Policy URL in the LinkedIn Developer app.

## Before using it publicly

Update the contact email in `index.html`:

```html
privacy@example.com
```

Replace it with your real support or privacy email.
